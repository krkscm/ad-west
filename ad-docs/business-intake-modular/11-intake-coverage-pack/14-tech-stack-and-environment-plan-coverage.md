# Tech Stack and Environment Plan Coverage

## Intake Stack Decisions
- Frontend: React and TypeScript.
- Backend: NestJS.
- Database: PostgreSQL.
- Cache and sessions: Redis.
- File storage: MinIO.
- Workflow and messaging automation: Deferred in current phase.
- Auth: JWT, password + captcha login, account lockout.
- Containerization: Docker Compose.

## Coverage Checklist
- [ ] Stack appears in design baseline.
- [ ] Stack appears in implementation plan.
- [ ] Stack appears in testing strategy.
- [ ] Operational handover references stack ownership.
