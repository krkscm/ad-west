# Business Intake Modular Pack

## Purpose
This folder decomposes business intake into small, maintainable modules that directly drive development phases.

## Best Practice Rules
- One concern per file.
- Stable IDs for traceability.
- Explicit in-scope and out-of-scope boundaries.
- Every module mapped to SDLC and acceptance docs.
